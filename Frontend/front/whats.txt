1) I have used Code Splitting Technique React.lazy() for downloading page when user wants to interact with them.

An Intro or Loading page — it will be visible until data comes.
3) I have used IntersectionObserver for loading data as per user requirement without reloading the page.
4) I have used useQuery library, or you can say dependency, for storing search results in the form of cache and preventing unnecessary list renders.
5) I have used forwardRef for sharing ref in other components for IntersectionObserver.
6) I have used useRef for tracking scroll position for movement.
